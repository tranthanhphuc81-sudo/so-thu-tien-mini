🔥 Kết nối Firebase (đồng bộ đa thiết bị)
⚡ Sau khi kết nối, NVTD và CTV đồng bộ real-time dù khác mạng, khác điện thoại. Hoàn toàn miễn phí.
Hướng dẫn: 1) Vào console.firebase.google.com → Tạo project → Realtime Database → Tạo database → Copy URL & API Key → Dán vào đây.
🔴 Chưa kết nối – Dùng bộ nhớ cục bộ
❌ Lỗi kết nối:
🛡️ Bảo mật Firebase (Bắt buộc)
⚠️ Thực hiện 2 bước này để bảo vệ dữ liệu khách hàng:
1. Firebase Console → Authentication → Sign-in method → Bật Email/Password → Save
2. Vào Realtime Database → Tab Rules → Xóa hết nội dung cũ → Dán JSON dưới đây → Bấm Publish 
{
  "rules": {
    ".read": false,
    ".write": false,
    "ctvAllowed": {
      ".read": true,
      ".write": "auth != null && auth.token.email === 'nvtd@sothu.app'"
    },
    "data": {
      ".read": "auth != null",
      ".write": "auth != null"
    }
  }
}
💡 Mật khẩu xác thực phía server (Firebase Auth) — không ai đọc được hash từ database.
